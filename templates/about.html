What this project does

The application periodically checks a set of radio station sources, normalizes the track metadata and stores detections in a local database. The web UI shows recent detections, charts and station statistics.

Tech: Python, Flask, SQLite (optionally MySQL), Bootstrap, simple JS for charts.

How to run
  • Start the background checker: python main.py
  • Open the web UI (default port 5000) or use the static pages in /static_web
  • See README and SETUP_PI.md for deployment details.
